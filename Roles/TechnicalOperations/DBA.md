# DBA

## Role Summary
The DBA provides database expertise and operational support across the business, while reporting into Production Operations. This role maintains SQL installs and database runtime health in hosted environments, optimises database performance, and partners with developers to produce efficient database design and architecture. The DBA also supports Production Operations by enabling safe customer merges and data transfers (e.g. advising on data movement approaches, performance impact, and database-side tooling).

## Skills
- Strong experience administering and operating SQL databases in production environments
- Experience with performance tuning, indexing strategies, and query optimisation
- Ability to work with developers to improve database schema and data access patterns
- Understanding of backup/restore, HA/DR concepts, and capacity planning
- Strong troubleshooting skills for database and data-related incidents
- Ability to automate database operations (scripts/tooling) where appropriate
