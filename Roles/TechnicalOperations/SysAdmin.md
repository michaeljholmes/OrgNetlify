### SysAdmin (this role)
The SysAdmin is the primary hands-on administrator within IT Operations.

This role specialises in day-to-day system administration work (server builds and
maintenance, OS patching, user/service administration, and operational support),
and is typically the first point of execution for IT Operations tasks.

## Skills

- Strong experience administering Windows Server and Linux systems
- Experience with Active Directory (or equivalent directory services) and access control
- Experience with OS patching, configuration management, and change control processes
- Understanding of infrastructure security, hardening, and vulnerability remediation
- Good understanding of networking fundamentals and firewall/VPN concepts
- Experience operating virtualised infrastructure and managing VM lifecycles
- Strong troubleshooting skills across OS, networking, and core IT services
- Ability to write automation and tooling scripts (e.g. PowerShell, Bash, Python)
- Ability to work independently and take ownership of IT-managed systems
