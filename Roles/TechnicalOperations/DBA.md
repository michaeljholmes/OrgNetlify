# DBA
## Role Summary
The Database Administrator (DBA) is responsible for the performance, integrity, and security of databases. This role involves designing, implementing, and maintaining database systems to support application and business needs.

**Scope & Accountability:** Owns database reliability, performance, security, and recoverability, ensuring data platforms support application and business needs safely.

## Responsibilities
- Install, configure, and maintain database management systems (SQL Server, PostgreSQL).
- Monitor database performance and implement optimisations (indexing, query tuning).
- Design and implement backup and recovery strategies.
- Ensure database security through access controls, encryption, and auditing.
- Plan and execute database migrations and upgrades.
- Collaborate with development teams on schema design and data modelling.
- Troubleshoot and resolve database-related issues.
- Maintain documentation for database configurations and procedures.

## Skills
- Strong experience with relational databases (SQL Server, PostgreSQL).
- Proficient in SQL and query optimisation techniques.
- Knowledge of database backup, recovery, and high availability solutions.
- Understanding of database security best practices.
- Experience with database monitoring and performance tuning tools.
- Familiarity with cloud database services (Azure SQL, AWS RDS) is a plus.

## Reports to
- Platform Engineering Lead
