## Skills
- Proficient with React and modern component patterns.
- Strong JavaScript/TypeScript skills, including typing patterns and safe refactoring.
- Solid HTML and CSS fundamentals (layout, responsive design) and experience with modern styling approaches.
- Accessibility awareness (semantic HTML, keyboard navigation, ARIA basics) and building inclusive UI.
- State management and data-fetching patterns (component state, global state where needed, caching).
- Experience integrating with APIs (REST, GQL), and handling loading, error states, and edge cases.
- Frontend testing practices (unit/component tests, contract and e2e testing).
- Performance awareness (bundle size, render performance) and using browser devtools to diagnose issues.

- Senior Frontend Engineer
