# Team Structure

This repository contains the organisation chart and role definitions for the team.

## Requirements

To view the Markdown files with inline content rendering (e.g., `@import` directives), you need:

- **VS Code** with the [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced) extension installed.

### Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search for **"Markdown Preview Enhanced"**
4. Click **Install**

### Usage

- Open any `.md` file
- Use **Ctrl+Shift+V** or right-click and select **"Markdown Preview Enhanced: Open Preview"** to view with inline imports rendered
